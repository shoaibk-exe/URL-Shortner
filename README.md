# Django URL Shortener 🔗

A simple URL shortening service built with Django.

## Features 🚀
- Shorten long URLs
- User authentication (signup/login)
- Bootstrap frontend
- SQLite/PostgreSQL database support

## Technologies Used 🛠️
- Python 3
- Django
- Bootstrap 5
- SQLite
